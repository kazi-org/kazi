   pinning test.

   zero-diff dispatch parked on its own background checks, #1550

   kazi-internal branches).
3. **The predicate-integrity feature family** (#1520 sealed predicates,
   #1521 controller-owned capture/evidence store, #1522 visual_judge
   provider, #1501 predicate quality metrics) -- all from observed gaming or
   observability gaps in consumer-project runs. #1520/#1521/#1522 form a
   dependency chain (seal the inputs, own the pixels, judge the pixels);
   each needs an ADR before code since all three add core loop/provider
   surface.

Acceptance: every issue above is closed citing a merged PR (or an ADR +
lands-later disposition explicitly recorded on the issue); no fix weakens an
existing assertion; docs land with the code (ADR-0034).
